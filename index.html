Petunjuk Deploy & Koneksi Database

Untuk menyambungkan frontend ini dengan Google Sheets menggunakan Google Apps Script:

  1. Buat Google Spreadsheet baru.
  2. Salin Spreadsheet ID dari URL spreadsheet Anda.
  3. Buat proyek Google Apps Script melalui menu Ekstensi di spreadsheet.
  4. Buat fungsi backend doGet() untuk menyajikan file HTML ini.
  5. Gunakan fungsi google.script.run dari frontend untuk mengirim/membaca data secara dinamis dari Sheets.
  6. Simpan project Apps Script, lalu lakukan Deploy sebagai Aplikasi Web (Web App).
Arsitektur landing page ini sudah disiapkan agar siap diintegrasikan secara instan kapan saja modul backend Google Apps Script Anda siap!
